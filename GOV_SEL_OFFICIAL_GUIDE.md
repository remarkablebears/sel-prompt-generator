## 📋 教育部SEL五大能力

根據教育部官方頒布的社會情緒學習中長程計畫，SEL包括五大核心能力：

### 1️⃣ 自我覺察 (Self-Awareness)
**定義**：認識自己的情緒、想法、價值觀與優勢，了解自己的學習風格

**在英語教學中的應用**：
- 學生能識別自己在英語學習中的情緒
- 了解自己的學習優勢與挑戰
- 認識個人的文化認同與價值觀

**教學策略**：
- 反思日誌與自評
- 情緒觀察練習
- 優勢識別活動

---

### 2️⃣ 自我管理 (Self-Management)
**定義**：管理個人情緒、設定目標、堅持克服挑戰，展現負責任的行為

**在英語教學中的應用**：
- 學生能管理在英語表達時的焦慮與害怕
- 設定現實且具挑戰性的語言學習目標
- 面對失敗時的堅持與恢復力

**教學策略**：
- 目標設定與追蹤
- 情緒管理技巧
- 失敗後的反思與成長

---

### 3️⃣ 社會覺察 (Social Awareness)
**定義**：理解他人觀點、展現同理心，認識社會議題與多元觀點

**在英語教學中的應用**：
- 學生能理解英語使用者的多元文化背景
- 展現對他人經歷與感受的同理心
- 認識全球社會議題（貧窮、性別平等、氣候等）

**教學策略**：
- 多元文化理解活動
- 同理心建立練習
- 全球議題討論

---

### 4️⃣ 人際關係技巧 (Relationship Skills)
**定義**：建立健康人際關係、有效溝通、團隊合作、協商與衝突解決

**在英語教學中的應用**：
- 學生能用英文進行有效的協商與對話
- 發展團隊合作與溝通技巧
- 解決衝突與處理不同意見

**教學策略**：
- 小組討論與協作
- 角色扮演與情境對話
- 衝突解決技巧練習

---

### 5️⃣ 負責任的決定 (Responsible Decision-Making)
**定義**：做出符合倫理的決定、考量他人影響、展現公民責任感

**在英語教學中的應用**：
- 學生能分析社會議題並做出負責任的決定
- 考量決定對他人與社區的影響
- 展現公民參與精神

**教學策略**：
- 倫理案例分析
- 決策模擬活動
- 社區服務融合

## 🎯 計畫核心願景

### 主要目標
✓ 營造友善且幸福的校園環境
✓ 提升教職員與學生的情緒健康
✓ 培養人際互動、同理關懷、道德責任等知識、技巧、態度、信念
✓ 實踐全人教育理念

### 五大推動面向

| 面向 | 說明 |
|------|------|
| **1. 研發與開展在地經驗** | 發展本土性SEL內涵，建立在地實踐案例 |
| **2. 人才培力** | 涵養教職員的SEL社會與情意知能，包括職前教師、現職教職員、學校領導人 |
| **3. 學校文化與環境** | 建立支持性團體、教學分享，營造正向校園文化 |
| **4. 跨系統連結** | 結合社區教育、家庭教育、非營利組織資源，整全式推動 |
| **5. 推廣傳播與國際連結** | 建立資源平臺，與國際SEL理論對話 |

## 💡 在英語教學中實踐SEL的原則

### 原則1：與十二年國教課綱對齊
✓ 符合英語文領綱的核心素養
✓ 支持「溝通互動」、「自主行動」、「社會參與」的發展
✓ 融入議題融滲教學

### 原則2：全人教育理念
✓ 不僅教授語言知識與技能
✓ 培養學生的情感、態度與價值觀
✓ 支持學生的整體發展

### 原則3：情緒安全優先
✓ 創造心理安全的學習環境
✓ 尊重學生個人邊界與差異
✓ 提供支持與鼓勵

### 原則4：真實與有意義
✓ 連結真實情境與生活經驗
✓ 運用真實的社會議題
✓ 促進學生實際應用

## 🎓 教學實踐建議

### 課程規劃階段
1. **確認SEL目標** - 選擇學生最需要發展的能力
2. **選擇教學主題** - 與學生生活相關的有意義話題
3. **設計學習經驗** - 提供多元的參與機會
4. **規劃評量方式** - 了解學生SEL能力的發展

### 教學實施階段
1. **營造安全環境** - 建立信任與心理安全感
2. **提供支架** - 循序漸進地支持學生
3. **引導反思** - 透過提問啟發學生思考
4. **鼓勵參與** - 給予學生勇敢嘗試的機會

### 評估與回饋階段
1. **形成性評量** - 觀察學生的SEL發展過程
2. **檔案評量** - 蒐集學生的成長足跡
3. **建設性回饋** - 提供具體的改進建議
4. **學生反思** - 促進學生自我評估與成長

## 🌟 教師常見問題

**Q: 這個工具是教育部官方的嗎？**
A: 此工具由台灣英語教師根據教育部官方發布的SEL中長程計畫開發，完全符合教育部的框架與願景。

**Q: 可以用在哪些課程？**
A: 可用於英語課程、綜合活動課、班會、導師時間，以及任何需要融合SEL的課程。

**Q: 如何確保符合十二年國教課綱？**
A: 生成的Prompt會提示您整合課綱核心素養，並支持「溝通互動」、「自主行動」、「社會參與」的發展。

**Q: 學生家長會支持嗎？**
A: 是的！教育部強調SEL是全人教育的實踐，與家長的期許一致。生成的Prompt可直接與家長溝通課程目標。

**Q: 如何評估SEL的實施成效？**
A: 此工具會引導您建立形成性評量、檔案評量與學生反思機制，符合教育部的評估建議。
