## 🎓 四種使用情景

### 情景1️⃣：我要快速規劃一堂課

**推薦工具**：SEL+SDG整合版 `index.html`

```
流程：
1. 打開 index.html
2. 選擇「教師用」→ 「課程規劃」
3. 填寫班級、年級、主題
4. 選擇1-2個SEL能力和SDG目標
5. 點「生成Prompt」
   ↓
6. 複製到ChatGPT
7. 獲得完整教案（15分鐘內）
```

**優勢**：最快、最直接、最全面

---

### 情景2️⃣：我想實踐教育部的幸福教育政策

**推薦工具**：教育部官方版本 `sel-gov-official.html` ⭐

```
流程：
1. 打開 sel-gov-official.html
2. 選擇「課程設計」或「活動設計」
3. 填寫班級與主題
4. 選擇5大SEL能力中的核心能力
5. 點「生成課程Prompt」
   ↓
6. 複製到Claude或ChatGPT
7. 獲得符合教育部中長程計畫的教案
```

**優勢**：
- ✅ 完全對齊教育部官方框架
- ✅ 融合五大推動面向
- ✅ 包含教育部聯絡資訊
- ✅ 適合學校行政呈報
- ✅ 符合國際趨勢（OECD、UNESCO）

---

### 情景3️⃣：我想深入探討某個SDG目標

**推薦工具**：SDG完整版 `sdg-complete.html`

```
流程：
1. 打開 sdg-complete.html
2. 選擇「課程設計」
3. 從17個SDG目標中選擇（如：SDG 5性別平等）
4. 填寫班級與時間
5. 點「生成課程Prompt」
   ↓
6. 獲得該目標的深度課程規劃
```

**適合**：
- 專題教學（如環境月、性平月）
- 跨領域合作設計
- 國際教育推廣

---

### 情景4️⃣：我想進行學生自主口說練習

**推薦工具**：SEL完整版 `sel-complete.html` 或官方版

```
流程：
1. 打開 sel-complete.html
2. 選擇「口說練習」
3. 輸入練習主題（如：情緒表達、解決衝突）
4. 選擇核心SEL能力
5. 點「生成練習Prompt」
   ↓
6. 複製給ChatGPT或Claude
7. 學生可獨立進行AI口說練習
```

**適合**：
- 課後自主學習
- 補救教學
- 課堂延伸練習
